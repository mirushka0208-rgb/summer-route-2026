🇬🇪 Тбилиси🌡️ ~26–30 °C · солнечно, жарко

  • Старый город + крепость Нарикала — канатка + пешком · ~2–3 ч
  • Серные бани Абанотубани — ~1–2 ч
  • Мцхета (Светицховели + Джвари) — с водителем · ~полдня

🇬🇷 Крит / Ханья🌡️ ~26–29 °C · солнечно, море ~23 °C

  • Старая венецианская гавань + маяк — пешком вечером · ~2 ч
  • Круиз Балос + Грамвуса — катер из Кисамоса · целый день ~7 ч
  • Elafonissi (розовый пляж) — тур из Ханьи · целый день

🇩🇰 Копенгаген🌡️ ~19–22 °C · мягко, длинный день

  • Nyhavn (цветная набережная) — пешком · ~1–2 ч
  • Концерт The Weeknd (Parken) — вечер 19.06
  • Тиволи (если время) — ~2–3 ч

🇫🇷 Париж🌡️ ~23–26 °C · тепло

  • Parc Astérix — ~45 мин от города · целый день
  • Cité des Enfants — для детей · ~полдня
  • Mont-Saint-Michel — поездом · целый день (~3 ч в одну)
  • Зоопарк Венсен — ~полдня

🇳🇴 Берген🌡️ ~16–18 °C · часто дождь

  • Брюгген (ганза, ЮНЕСКО) — пешком · ~1 ч
  • Фуникулёр Флёйбанен на Флёйен — виды + хайк · ~1.5–2 ч
  • Рыбный рынок — ~30–45 мин

Город реально осмотреть за полдня.

Флом🌡️ ~16–19 °C · переменно

  • Фломсбана (горный поезд) + круиз по Нерёй-фьорду — ~полдня
  • Смотровая Стегастейн — на авто · ~1.5 ч

Сольворн / Люстра-фьорд🌡️ ~17–20 °C

  • Ставкирка Урнес (ЮНЕСКО, паромчик) — ~2 ч
  • Фьорд / Walaker — прогулка · ~1 ч

Гейрангер🌡️ ~16–19 °C · переменно

  • Круиз по Гейрангер-фьорду («Семь сестёр») — ~1.5 ч
  • Смотровая Dalsnibba (1500 м) — на авто · ~1.5 ч
  • Орлиная дорога (Ørnesvingen) — ~30 мин
  • Хайк к Storseterfossen (за водопадом) — ~2.5 ч

На круиз + смотровые нужен почти целый день.

Олесунн🌡️ ~15–18 °C · ветрено

  • Подъём на г. Аксла (418 ступеней) — виды на город · ~1 ч
  • Центр в стиле ар-нуво — пешком · ~1.5 ч

Ставангер / Прейкестулен🌡️ ~16–18 °C

  • Прейкестулен (Pulpit Rock) — хайк · ~4 ч туда-обратно (+трансфер/паром)
  • Гамле-Ставангер (белые домики) — ~1 ч

🇮🇸 Рейкьявик🌡️ ~11–13 °C · ветрено, свет ~20 ч

  • Hallgrímskirkja + лифт на башню — ~1 ч
  • Голубая лагуна (по пути) — бронь заранее · ~2–3 ч
  • Киты из старой гавани — ~3 ч

Snæfellsnes🌡️ ~10–12 °C

  • Kirkjufell + водопад Kirkjufellsfoss — ~1 ч
  • Тюлени Ytri-Tunga — ~30 мин
  • Arnarstapi–Hellnar (прибрежная тропа) — ~1.5 ч

Золотое кольцо / Hella🌡️ ~12–14 °C

  • Þingvellir (рифт, ЮНЕСКО) — ~1.5 ч
  • Geysir / Strokkur (бьёт ~каждые 7 мин) — ~45 мин
  • Gullfoss (водопад) — ~45 мин
  • Secret Lagoon — ~1.5 ч

Классика на целый день.

Вик🌡️ ~10–12 °C · ветрено

  • Чёрный пляж Рейнисфьяра + базальт — ~1 ч
  • Мыс Дюрхолаэй (тупики) — ~45 мин
  • По пути: Сельяландсфосс + Скоугафосс — по ~30–45 мин

Хёбн / Йёкюльсаурлоун🌡️ ~10–12 °C

  • Лагуна Йёкюльсаурлоун + лодка — ~1.5 ч
  • Diamond Beach (льдины на чёрном песке) — ~30 мин
  • Skaftafell / водопад Svartifoss (хайк) — ~2.5 ч

Mývatn🌡️ ~11–14 °C · север мягче

  • Hverir (фумаролы, грязевые котлы) — ~45 мин
  • Krafla / кратер Víti — ~1 ч
  • Mývatn Nature Baths — ~1.5 ч
  • Goðafoss (по пути) — ~30 мин

Акюрейри🌡️ ~13–15 °C · часто теплее всего

  • Киты из Хусавика (рядом) — ~3 ч
  • Центр + Akureyrarkirkja + ботсад — ~1.5 ч
  • Goðafoss — ~30 мин

🇩🇪 Франкфурт🌡️ ~24–27 °C · тепло

  • Рёмерберг (старая площадь) — ~1 ч (если есть время до рейса)
